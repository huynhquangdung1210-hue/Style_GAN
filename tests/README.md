# Style Transfer API Test Scripts

A collection of test scripts to validate the style transfer API functionality.

## Quick Test

```bash
python test_api.py
```

## Load Testing

```bash
python load_test.py --concurrent-users 10 --duration 60
```

## Files

- `test_api.py`: Basic API functionality test
- `load_test.py`: Performance and load testing
- `test_images/`: Sample images for testing
